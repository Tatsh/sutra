static method RFC3339ToTimestamp (line 73)

Convert an RFC3339 (HTML 5 version) timestamp to UNIX. Timezone is ignored.

HTML 5 mandates 2 extra constraints:

  • the literal letters T and Z in the date/time syntax must always be uppercase
  • the date-fullyear production is instead defined as four or more digits representing a number greater than 0

  • return: UNIX timestamp or boolean FALSE. If FALSE is returned, the string was invalid.
  • throws: fValidationException If $throw is set to TRUE and the timestamp is invalid.
  • access: public
static integer|boolean RFC3339ToTimestamp (string $rfc, [boolean $throw = FALSE])
  • string $rfc: The RFC value, like: 1990-12-31T23:59:60Z or 1996-12-19T16:39:57-08:00.
  • boolean $throw: Throw an fValidationException if the timestamp is is invalid. Defaults to FALSE.
