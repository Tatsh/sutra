		static method printSortableColumn (line 261)
	
 
	
	

Prints a sortable column header `a` tag


The a tag will include the CSS class `'sortable_column'` and the  direction being sorted, `'asc'` or `'desc'`.
{{{  #!php  fCRUD::printSortableColumn('name', 'Name');  }}}
would create the following HTML based on the page context
{{{  #!html  <!-- If name is the current sort column in the asc direction, the output would be -->  <a href="?sort=name&dir=desc" class="sorted_column asc">Name</a>
<!-- If name is not the current sort column, the output would be -->  <a href="?sort-name&dir=asc" class="sorted_column">Name</a>  }}}
