    registerFormatCallback(callback $callback) : void
    Static

    The callback should accept two parameters: - $value: the string value of the number - $remove_zero_fraction: a boolean indicating if a zero fraction should be removed

    The callback should return a string, the formatted $value.

    Parameters

    $callback

    callback

    The callback to pass the fNumber value to - see method description for parameters

    Allows setting a callback to clean any formatted values so they can be properly parsed - useful for languages where `,` is used as the decimal point
