Extension to Flourish fJSON class to provide methods for JSONP support.

This class is a compatibility class for the [http://php.net/json json extension] on servers with PHP 5.0 or 5.1, or servers with the json extension compiled out.

This class will handle JSON values that are not contained in an array or object - such values are not valid according to the JSON spec, but the functionality is included for compatiblity with the json extension.
