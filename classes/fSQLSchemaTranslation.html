parseSQLiteColumnDefinitions(string $sql) : array

The return value is an associative array with the keys being column names and the values being arrays containing the following keys: - definition: (string) the complete column definition - pieces: (array) an associative array that can be joined back together to make the definition - beginning - column_name - data_type - not_null - null - default - unique - primary_key - check_constraint - foreign_key - deferrable - comment/end

Parameters

$sql

string
