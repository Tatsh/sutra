    Add a callback that will be used in place of the ones here.

    addCallback(string $locale, string $method_name, string | array $callback) : void

    For use with different languages and locales.

    All callbacks must receive one value, an integer, and return a string.

    Parameters

    $locale

    string

    The locale name. Should be a standard locale name such as en-GB, fr-FR, etc.

    $method_name

    string

    Method name in this class to override. One of: ordinal, ordinalSuffix. The instance methods will also use this callback.

    $callback

    stringarray

    Callback to use.

    Exceptions

    \fProgrammerException If the method name or locale is invalid.
