Static

The message will be printed in a p tag if it does not contain any block level HTML, otherwise it will be printed in a div tag.

Parameters

$name

mixed

The name or array of names of the message(s) to show, or '*' to show all

$recipient

string

The intended recipient

$css_class

string

Overrides using the $name as the CSS class when displaying the message - only used if a single $name is specified

Returns
