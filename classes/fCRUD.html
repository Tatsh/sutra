    printSortableColumn(string $column, string $column_name) : void
    Static

    The a tag will include the CSS class 'sortable_column' and the direction being sorted, 'asc' or 'desc'.

    {{{

    !php

    fCRUD::printSortableColumn('name', 'Name'); }}}

    would create the following HTML based on the page context

    {{{

    !html

    Name

    Name }}}

    Parameters

    $column

    string

    The column to create the sortable header for

    $column_name

    string

    This will override the humanized version of the column
