    setTemplate(string $template_name) : \sCRUDForm

    All input field HTML will be passed as variables to the template by their column name.

    No containers or labels are generated when templating is used.

    The form attributes will be passed as variable $form_attr to the template.

    The buttons will be passed as an array $buttons with each piece being the button HTML.

    note Requires sTemplate.

    Parameters

    $template_name

    string

    Template name.

    Returns
